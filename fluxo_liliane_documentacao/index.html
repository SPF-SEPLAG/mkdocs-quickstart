Fluxo Power Automate – Integração de Planilha ISSQN e Registro de Códigos de Barras

Objetivo

Este fluxo automatiza a extração de dados filtrados da planilha CONTROLE DCF 2024.xlsx (navegador) para a planilha local planilha_controle_issqn.xlsx, e, em seguida, realiza o registro de cada item no SIAFI via aplicação InputMinot, incluindo a captura e inserção do código de barras da guia de ISSQN.

Documentação

O fluxo principal chama, em sequência, dois subfluxos:

  1. Planilha Liliane
  2. Registrar Código de Barras Liliane

Subfluxo: Planilha Liliane

  1. Lançar Chrome
  2. Ação: WebAutomation.LaunchChrome.LaunchChrome
  3. Parâmetros:
    • Url: =vUrlPlanilhaDcf
    • WindowState: Maximizado
    • ClearCache: False
    • ClearCookies: False
    • BrowserInstance ⇒ vPlanilhaDcf

  4. O que faz: Esta etapa inicializa o navegador Chrome e abre a planilha online “CONTROLE DCF 2024.xlsx” usando a URL configurada na variável vUrlPlanilhaDcf. Permite ao fluxo acessar e interagir com a planilha em ambiente web.

  5. Anexar à aba da planilha

  6. Ação: WebAutomation.LaunchChrome.AttachToChromeByTitle
  7. Parâmetros:
    • TabTitle: CONTROLE DCF 2024.xlsx
    • BrowserInstance ⇒ PlanilhaDcfChrome

  8. O que faz: Conecta o fluxo à aba já aberta no Chrome cujo título corresponde à planilha de controle, garantindo que as ações seguintes sejam direcionadas ao navegador correto.

  9. Solicitar seleção de dados

  10. Ação: Display.ShowMessageDialog.ShowMessage
  11. Parâmetros:
    • Title: Atenção!
    • Message: Instruções para aplicar filtros e selecionar registros, clicar em A2 e confirmar.
    • Icon: Warning

  12. O que faz: Exibe uma caixa de diálogo com instruções para o usuário aplicar filtros na planilha web e selecionar manualmente os dados de interesse antes de prosseguir.

  13. Selecionar e copiar registros

  14. Ações:
    • MouseAndKeyboard.SendKeys.FocusAndSendKeysByInstanceOrHandle ⇒ Seleção (Shift+Ctrl+Down e Right×20)
    • MouseAndKeyboard.SendKeys.FocusAndSendKeysByInstanceOrHandle ⇒ Ctrl+C

  15. O que faz: Seleciona a coluna de dados filtrados na planilha web a partir da célula A2, abrange as colunas necessárias e copia o conteúdo para a área de transferência.

  16. Abrir planilha local

  17. Ação: Excel.LaunchExcel.LaunchAndOpenUnderExistingProcess
  18. Parâmetros:
    • Path: %InputFolderPath%\planilha_controle_issqn.xlsx
    • Visible: True
    • Instance ⇒ PlanilhaExcel

  19. O que faz: Abre no Excel a planilha local planilha_controle_issqn.xlsx a partir da pasta definida em InputFolderPath, deixando-a visível para receber os dados copiados.

  20. Posicionar e ativar célula

  21. Ações:
    • MouseAndKeyboard.SendKeys ⇒ {Down}
    • Excel.ActivateCellInExcel ⇒ Coluna A, Linha 2

  22. O que faz: Move o cursor para a primeira célula livre (A2) na planilha local, preparando o local de colagem dos dados.

  23. Formatar Texto para Colunas e colar dados

  24. Ações:
    • MouseAndKeyboard.SendKeys ⇒ Atalho do assistente de Texto para Colunas (Ctrl+T, Alt+C, K, L)
    • Ajuste de direção com {Left}
    • MouseAndKeyboard.SendKeys ⇒ Ctrl+V

  25. O que faz: Executa o assistente de “Texto para Colunas” para separar corretamente os campos copiados, ajusta o delimitador e cola os dados formatados na planilha.

  26. Salvar e fechar planilha

  27. Ação: Excel.CloseExcel.CloseAndSave
  28. O que faz: Salva as alterações na planilha local e fecha a instância do Excel, garantindo que os dados estejam gravados no arquivo.

Subfluxo: Registrar Código de Barras Liliane

  1. Abrir planilha local
  2. Ação: Excel.LaunchExcel.LaunchAndOpenUnderExistingProcess
  3. Parâmetros:
    • Path: %InputFolderPath%\planilha_controle_issqn.xlsx
    • Visible: False
    • Instance ⇒ Planilha

  4. O que faz: Abre a planilha de controle em modo invisível para leitura dos dados já formatados.

  5. Obter primeira célula livre

  6. Ação: Excel.GetFirstFreeColumnRow
  7. Saídas:
    • FirstFreeColumn ⇒ ColLivre
    • FirstFreeRow ⇒ LiLivre

  8. O que faz: Identifica a primeira coluna e linha vazias na planilha, delimitando o intervalo de dados existente para leitura.

  9. Ler dados para tabela

  10. Ação: Excel.ReadFromExcel.ReadCells
  11. Intervalo: A1 até ColLivre-1 / LiLivre-1
  12. Parâmetros:
    • FirstLineIsHeader: True
    • Saída ⇒ DadosIssqn

  13. O que faz: Lê todos os registros existentes na planilha local e armazena-os em uma variável de tabela (DadosIssqn) para processamento.

  14. Fechar planilha local

  15. Ação: Excel.CloseExcel.Close

  16. O que faz: Fecha o Excel sem salvar, pois não houve alteração no arquivo nesta etapa.

  17. Inicializar contagem de linha

  18. Ação: SET SheetLineCount TO 2

  19. O que faz: Define o contador inicial de linha (SheetLineCount) como 2, indicando que o processamento começará a partir da segunda linha da tabela de dados.

  20. Conectar à planilha web

  21. Ação: WebAutomation.LaunchChrome.AttachToChromeByTitle
  22. Parâmetros:
    • TabTitle: CONTROLE DCF 2024.xlsx
    • BrowserInstance ⇒ PlanilhaDcfChrome

  23. O que faz: Reconecta o fluxo à aba da planilha web aberta anteriormente, pronta para informar o código de barras.

  24. Solicitar seleção da célula de OP

  25. Ação: Display.ShowMessageDialog.ShowMessage
  26. Parâmetros: instruções para aplicar filtros e selecionar a primeira célula vazia da coluna de OP.

  27. O que faz: Exibe instruções ao usuário para filtrar e selecionar manualmente a célula onde será registrada a OP no ambiente web.

  28. Carregar tabelas de contas

  29. Ações:
    • Variables.GenerateDataTableFromCSV ⇒ Contas1501
    • Variables.GenerateDataTableFromCSV ⇒ ContasOutrasUos

  30. O que faz: Importa tabelas auxiliares de contas orçamentárias a partir de arquivos CSV, usadas nas etapas de cálculo e conferência de conta.

  31. Inicializar variáveis de exceção

  32. Ações: SET NumberException, LengthException, FloatException, RangeException, SameDayException, WeekendException para ''

  33. O que faz: Reinicia variáveis de controle de erro, assegurando que eventuais exceções sejam tratadas corretamente durante o loop.

  34. Loop de processamento para cada registro

    • Descrição: Itera sobre cada linha de DadosIssqn, extraindo informações necessárias e informando-as no sistema SIAFI via InputMinot.
    • Passos principais dentro do loop:
    • ID: SET ID TO LiDadosIssqn['ID']
    • Processo SEI: Scripting.RunPythonScript (substituir / por .) ⇒ ProcessoSEI
    • Número da NF: SET NumeroNF TO LiDadosIssqn['Número da Nota Fiscal']
    • CNPJ: Scripting.RunPythonScript (remover pontuação) ⇒ CnpjPref
    • Prefixo Município: Scripting.RunPythonScript (tratar acentuação e nome) ⇒ Pref
    • Dados Orçamentários:
      • GmiFp e Fp via script Python
      • ValorIssqn via script Python de conversão
      • Guia
    • Subfluxos auxiliares:
      • CALL 'Extrair Uo'
      • CALL 'Informar Código de Barras'
      • CALL 'Conferir Conta'
      • CALL 'Informar Multa'
      • CALL 'Informar Linha'
      • CALL 'Informar Data Pagamento'
      • CALL 'Informar Op'
    • Interações com SIAFI (InputMinot):
      • MouseAndKeyboard.SendKeys para preencher campos em sequência
      • Display.ShowMessageDialog para conferência de texto
      • MouseAndKeyboard.SendKeys {F5} (simulação) e desativação da execução real
    • O que faz: Em cada iteração, o fluxo coleta dados da planilha local, processa informações orçamentárias via scripts Python, e preenche automaticamente os campos no sistema SIAFI, incluindo a inserção do código de barras e demais detalhes.

Passos Para Execução

  1. Pré-requisitos
  2. Instalar Power Automate Desktop e efetuar login corporativo
  3. Definir as variáveis de ambiente:
    • vUrlPlanilhaDcf: URL da planilha web
    • InputFolderPath: caminho da pasta local onde está planilha_controle_issqn.xlsx

  4. O que faz: Garante que todos os componentes necessários estejam configurados antes de iniciar o fluxo.

  5. Ambiente

  6. Ter a planilha CONTROLE DCF 2024.xlsx aberta no Chrome
  7. Garantir que planilha_controle_issqn.xlsx esteja em %InputFolderPath%
  8. Disponibilizar os arquivos CSV para contas: InputContas1501Csv, InputContasOutrasUosCsv

  9. O que faz: Prepara o ambiente de execução com os arquivos e instâncias corretas.

  10. Execução

  11. Executar o subfluxo Planilha Liliane
  12. Executar o subfluxo Registrar Código de Barras Liliane
  13. Seguir as instruções das caixas de diálogo para seleção de células, inserção de códigos de barras e conferências
  14. O que faz:Executa de forma sequencial os dois subfluxos, automatizando completamente o processo de extração, formatação e registro dos dados no sistema.
