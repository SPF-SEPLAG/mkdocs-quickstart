    Fluxo Power Automate – Liquidação

    Objetivo

    Este fluxo automatiza a liquidação, registrando no sistema SIAFI os dados das notas fiscais a serem liquidadas.

    Documentação

    O fluxo é composto por um fluxo principal (Main) e três subfluxos: Inicializar Variáveis de Fluxo, Executar Siafi Terminal e Siafi Web.

    1. O fluxo principal (Main) inicializa as variáveis chamando o subfluxo Inicializar Variáveis.
    2. Após a inicialização, o fluxo define a variável FilePath, que armazena o caminho da planilha auxiliar (dados.xlsx) localizada no diretório especificado. Esse arquivo contém os dados necessários para o processo de liquidação no SIAFI.
    3. Após acessar a planilha auxiliar, o fluxo define quatro variáveis essenciais para o processo:
    4. Minot: armazena a janela em instância do SIAFI aberta.
    5. CnpjInss: contém o CNPJ do INSS.
    6. Uo: guarda o número da unidade orçamentária (padrão: 1501).
    7. RowIndex: armazena o valor 1, utilizada para navegação na planilha.
    8. Após a definição das variáveis, o fluxo inicia o Excel usando FilePath e obtém a primeira coluna e linha livre, armazenando-as nas variáveis ColunaLivre e LinhaLivre.
    9. O fluxo lê os dados da planilha do intervalo A1 até a última célula não vazia, armazenando-os em DadosLiquidacao.
    10. O fluxo itera sobre cada linha em DadosLiquidacao (loop for each), armazenando a linha atual em Linha.
    11. Para cada Linha, o fluxo envia os dados para o SIAFI (instância Minot), navegando no sistema.
    12. Condições IF:
    13. Se Linha["Conformidade"] ≠ N/A, lança o valor; caso contrário, pula com TAB.
    14. Verifica se Linha["Valor INSS"] e Linha["Valor ISSQN"] estão vazias:
    15. Se vazias: preenche com s; senão: preenche com n.
    16. Obtém a data atual (DataAtual) e formata como mês de competência.
    17. Converte o valor da NF em texto sem separador decimal.
    18. Define Ordenador com o valor de Linha["MASP"].
    19. Remove dígito final de MASP se iniciar com m e envia para o SIAFI.
    20. Se Valor INSS não está vazio, formata e armazena em ValorInss e CompetenciaInss.
    21. Verifica Valor ISSQN: se não vazio, formata e armazena em ValorIssqn. Valida CNPJ Município, interrompendo com erro se ausente ou tratando e armazenando em CnpjIssqn.
    22. Lógica composta:
      • Ambos INSS e ISSQN: envia CnpjInss, CompetenciaInss, ValorInss, ValorIssqn.
      • Apenas ISSQN: envia CnpjIssqn, ValorIssqn.
      • Apenas INSS: envia CnpjInss, CompetenciaInss, ValorInss.
    23. Lança Nome Credor, Número Nota Fiscal, ID e Processo SEI, removendo barras / se presentes.
    24. Formata Processo SEI, removendo / e envia.
    25. Envia F5 para realizar liquidação.
    26. Copia o conteúdo da tela (Ctrl + A, Ctrl + C) para SeqLiquidacao, processa com Python e atualiza.
    27. Reabre a planilha e grava SeqLiquidacao na coluna U, usando RowIndex.
    28. Envia F3 três vezes para retornar ao início.
    29. Após concluir as etapas no terminal do SIAFI, o fluxo inicia as etapas do SIAFI Web.
    30. Abre a planilha auxiliar (FilePath) no Power Automate.
    31. Lê os dados em DadosLiquidacao novamente.
    32. Itera sobre cada linha em DadosLiquidacao.
    33. Abre o Google Chrome e acessa o portal SIAFI Web (instância SiafiBranco).
    34. Procura pelo texto "Login Certificação Digital" e navega até os campos de login.
    35. Preenche as credenciais com UsuarioSiafi e SenhaSiafi.
    36. Clica em "Consulta / Impressão Documento Específico".
    37. Executa JavaScript para selecionar a Unidade Executora (elemento "UE").
    38. Navega com TAB e Down para preencher No.Empenho e Seq.Liquidação.
    39. Pressiona Ctrl + S para salvar o relatório (GeraRelatorioEmpenhoServlet).
    40. Na janela de diálogo, define o caminho usando FilePath + "\\liquidação.pdf" e confirma com Enter.
    41. Fecha as janelas do SIAFI Web e do gerador de relatórios.
    42. Abre nova instância do Chrome e acessa o portal SEI (variável Sei).
    43. Faz login no SEI com SeiSenha.
    44. Clica na barra de pesquisa e busca Processo SEI (valor em Linha).
    45. Pressiona Enter para localizar o processo.
    46. Clica em "Incluir Documento".
    47. Seleciona "Externo", preenche dados (DataAtual, Nº Empenho, Seq. Liquidação, Id) e realiza upload.
    48. Exclui o arquivo local após inserção no SEI.

    Passos Para Execução do Fluxo no Power Automate Desktop

    Pré-requisitos
    - Power Automate Desktop instalado
    - Fluxo criado
    - Planilha auxiliar (dados.xlsx) configurada na variável FilePath
    - Siafi Terminal instalado

    Fluxo de execução
    1. Fazer download do Power Automate Desktop (site da Microsoft).
    2. Instalar e fazer login com conta corporativa (CAMG).
    3. Criar novo fluxo (caso não exista):
    - Criar fluxo
    - Nomear fluxo
    - Criar subfluxos
    4. Colar os arquivos Main.txt, Inicializar variáveis de Fluxo.txt, Executar Siafi Terminal.txt e Executar Siafi Web.txt nos subfluxos correspondentes.
    5. Criar a planilha auxiliar com as colunas necessárias e definir seu caminho em FilePath.
    6. Executar fluxo.
